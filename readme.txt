== Installation ==

1. In your wordpress admin, Under the heading plugins click on Add New.
2. Search Joleado Live Chat Software in Search box.
3. Locate Joleado Live Chat Software plugins in the search results.
4. Click on Install Now Button.
5. After that Click on Activate Button to activate your plugin.
4. See the "Joleado chat Settings" menu on the left bar of your Wordpress admin for setup.
